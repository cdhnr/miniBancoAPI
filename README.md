# Descrição

API REST desenvolvida em **Spring Boot** para simular operações básicas de um sistema bancário.  
O projeto tem como objetivo aplicar conceitos de **Java**, **Spring Boot**, **JPA/Hibernate**, e boas práticas de arquitetura em APIs RESTful.
